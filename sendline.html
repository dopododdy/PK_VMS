📖 วิธีสร้าง LINE Notify Token
  1. เปิดเบราว์เซอร์ ไปที่ notify-bot.line.me
  2. กดปุ่ม "เข้าสู่ระบบ" ด้วยบัญชี LINE ของคุณ
  3. กดที่ "สร้าง Token" ที่หน้า My Page
  4. ตั้งชื่อ Token และเลือกห้องแชทที่ต้องการรับการแจ้งเตือน (เช่น "1-on-1 Chat with LINE Notify" หรือห้อง Group)
  5. คัดลอก Token ที่ได้รับ แล้วนำมาวางในช่อง Token ด้านบน
  6. กด "บันทึก Token" แล้วทดสอบส่งข้อความ
⚠️ Token จะถูกเก็บไว้ใน localStorage ของเบราว์เซอร์นี้เท่านั้น ไม่ได้ส่งไปเก็บที่เซิร์ฟเวอร์ใด
