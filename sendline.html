LINE Messaging API

คลินิกรักษาสัตว์หมอปิติพัฒน์ — LINE Official Account

ส่งข้อความผ่าน LINE Official Account ของคลินิกด้วย Messaging API — รองรับการส่งหาผู้ใช้เฉพาะราย (Push) และส่งหาผู้ติดตามทั้งหมด (Broadcast)

🔑 Channel Access Token

ดู Token ได้ที่ LINE Developers Console → เลือก Channel → Messaging API → Issue หรือ Copy Channel Access Token

✅ บันทึกแล้ว
📡 รูปแบบการส่ง

ดู User ID ได้จาก LINE Official Account Manager → ข้อมูลผู้ติดตาม หรือผ่าน Webhook event (follow / message) จาก LINE Platform

✏️ ข้อความ

0 / 5000 ตัวอักษร

📋 ผลการส่ง
📖 วิธีตั้งค่า LINE Messaging API
  1. เปิด developers.line.biz แล้วเข้าสู่ระบบด้วยบัญชี LINE
  2. เลือก Provider แล้วกด Create a new channel เลือก Messaging API
  3. กรอกข้อมูล Channel (ชื่อ / ประเภทธุรกิจ / อีเมล) แล้วกด Create
  4. ไปที่แท็บ Messaging API → เลื่อนลงหา Channel access token กด Issue แล้วคัดลอก Token
  5. วาง Token ในช่องด้านบน แล้วกด บันทึก Token
  6. ในโหมด Push ให้ระบุ LINE User ID ของผู้รับ (ขึ้นต้นด้วย U ตามด้วยตัวเลขและตัวอักษร 32 หลัก)
  7. พิมพ์ข้อความ หรือเลือกเทมเพลต แล้วกด ส่งข้อความ
💡 ระบบนี้ส่งข้อความผ่านเซิร์ฟเวอร์ท้องถิ่น (localhost:8080) เพื่อหลีกเลี่ยงข้อจำกัด CORS — กรุณาตรวจสอบว่า VASD Agent (app.py) กำลังทำงานอยู่
⚠️ Token จะถูกเก็บไว้ใน localStorage ของเบราว์เซอร์นี้เท่านั้น ไม่ได้ส่งไปเก็บที่เซิร์ฟเวอร์ภายนอกใด
